Semesterplan für's Studium als Kalender

Gib' her!

Statt jedes mal mühsam deine Termine aus der ABB_Gesamtplan_Moodle.xls-Tabelle und deinem Ausbildungsplan rauszusuchen gibt's das hier ganz einfach in einem Kalender.

Den Kalender hier kannst du auch in deinen (persönlchen) Kalender aufnehmen, dazu mehr in der Hilfe.

Deine Termine aus dem Semsterplan von der Moodle liegen schon vor, ich habe aber keinen Zugriff auf alle Ausbildungspläne, daher würde ich mich freuen wenn du mir den Ausbildungsplan deiner Klasse zukommenlassen könntest.

Ohne JavaScript kannst du dir den Kalender leider nicht anschauen. Du kannst dir aber hier alle .ics-Dateien ansehen. Wenn du deine Datei gefunden hast, kannst du sie aber trotzdem wie weiter unten bei Verwendung beschrieben verwenden :)

    Verwendung

    Semesterplan im Kalender anzeigen

    1. Auf der Startseite den "Gib' her!"-Button drücken
    2. Im "Klasse"-Auswahlfeld deine Klasse auswählen
    3. Dein Kalender wird jetzt angezeigt :)

    Semesterplan als Kalender abonnieren

    1. Auf der Startseite den "Gib' her!"-Button drücken
    2. Im "Klasse"-Auswahlfeld deine Klasse auswählen
    3. Dein Kalender wird jetzt angezeigt :)
    4. Auf klicken
    5. Rechtsklick auf "Kalender herunterladen"
    6. Adresse kopieren / Ziel kopieren
    7. Oben unter "Hilfe" deine bevorzugte Anwendung raussuchen und dort fortsetzen.

    Outlook

    Für Outlook kannst du einfach den "webcal öffen"-Link nutzen.

    Sollte das nicht funktionieren:

    • Auf klicken
    • Rechtsklick auf "Kalender herunterladen"
    • Adresse kopieren / Ziel kopieren
    • Outlook und dann den Kalender öffnen
    • Kalender → Hinzufügen → Online/Internet
    • Hier die oben kopierte Adresse einfügen
    • Fertig.

    Je nach Spracheinstellungen und Outlook-/Office-Version können die jeweiligen Punkte anders beschriftet sein.

    Google Calendar

    Bitte nutze folgende Schritte um den Kalender in deinen Google Kalender hinzuzufügen:

    • Auf klicken
    • Rechtsklick auf "Kalender herunterladen"
    • Adresse kopieren / Ziel kopieren
    • Google Calendar öffnen.
    • Auf das Dreieck nach unten bei "Andere Kalender" klicken und "Von URL hinzufügen" wählen.
    • Link von oben einfügen und "Hinzufügen" klicken.
    • Fertig.

    Je nach Spracheinstellungen können die jeweiligen Punkte anders beschriftet sein.

    Solltest du eine Möglichkeit gefunden haben das automatisch zu machen, öffne bitte ein GitLab Issue oder schreib mir eine Mail.

    Thunderbird mit Lighning

    Bitte nutze folgende Schritte um den Kalender in dein Thunderbird hinzuzufügen:

    • Auf klicken
    • Rechtsklick auf "Kalender herunterladen"
    • Adresse kopieren / Ziel kopieren
    • Thunderbird und dann Kalender öffnen.
    • Rechtsklick auf die Kalenderlist und "Hinzufügen" wählen.
    • Link von oben einfügen und "Hinzufügen" klicken.
    • Fertig.

    Je nach Spracheinstellungen können die jeweiligen Punkte anders beschriftet sein.

    Solltest du eine Möglichkeit gefunden haben das automatisch zu machen, öffne bitte ein GitLab Issue oder schreib mir eine Mail.

    Evolution

    Bitte nutze folgende Schritte um den Kalender in dein Evolution hinzuzufügen:

    • Auf klicken
    • Rechtsklick auf "Kalender herunterladen"
    • Adresse kopieren / Ziel kopieren
    • Oben links "Hinzufügen" und dann "Kalender" wählen.
    • Bei Typ "Web" auswählen.
    • Link von oben einfügen und "Hinzufügen" klicken.
    • Aktualisierung auf gewünschte Zeit einstellen. (1x je Woche sollte genügen)
    • ggf. "Sichere Verbindung" aktivieren. (empfohlen)
    • Restliche Felder ignorieren.
    • Fertig.

    Je nach Spracheinstellungen können die jeweiligen Punkte anders beschriftet sein.

    Windows Phone

    Um den Kalendar auf deinem Windows Phone zu nutzen, ist es das einfachste ihn in deinen Live Kalender hinzuzufügen. Er sollte dann in der Kalender-App erscheinen.

    Vorrausgesetzt du hast dein Windows Phone mit deiner Live ID verknüpft.

    Bitte nutze folgende Schritte um ihn hinzuzufügen:

    • Auf klicken
    • Rechtsklick auf "Kalender herunterladen"
    • Adresse kopieren / Ziel kopieren
    • calendar.live.com aufrufen (öffnet in neuem Tab/Fenster) und ggf. anmelden.
    • Oben in der Leiste "Importieren" wählen
    • Links auf "Abbonieren" klicken.
    • Link von oben in "Kalender URL" einfügen und einen Namen vergeben.
    • "Abbonieren" klicken.
    • Fertig. Der Kalender sollte jetzt in der Kalender-App angezeigt werden.

    Je nach Spracheinstellungen können die jeweiligen Punkte anders beschriftet sein.
